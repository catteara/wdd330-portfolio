Chapter 9: The Window Object

  • The Browser Object Model (BOM)
    • A collection of properties and methods that contain information about the browser and computer screen
    • In a browser environment that global object is the window object
    • Dialogs
      • Alert(), confirm(), prompt()
  • Browser History
    • Window.history can be used to access the information about any previously visited pages in the current browser session
    • Add .length and you can see how many pages have been visited before the current page.
    • Add .go(#) to the original statement and you can navigate to other viewed pages.
  • Controlling Windows
    • Window.open opens another window, for example a pop up
    • Window.screen object contains information about the screen the browser is displayed on 
  • Cookies
    • Small files saved locally
    • They can only be accessed by a web page with the same domain that set them.
    • Create a cookie
      • Document.cookie = …
  • Timing Functions
    • SetTimeout()
    • SetInterval()

Chapter 14: HTML5 APIs

  • Data-attribute
    • A way of embedding data in a web page using custom attributes that are ignored by the browser
    • Private on the page, so not intended to be used by an external service.
    • Attributes must be named using following format
      • Start with “data-“
      • Contain only lowercase letters, numbers, hyphens, dots, colons, or underscores
      • Include an optional string value
    • Provide a convenient way of adding data directly into the HTML markup
  • HTML APIs- 
    • Web Storage
    • Geolocation
    • Web Workers
  • Drawing Shapes in Canvas
    • Accessing the canvas element in JavaScript
  • Shims and Polyfills
    • Libraries of code that allow you to use APIs as usual. They then fill in the necessary code that’s not provided natively by the user’s browser.
